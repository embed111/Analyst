2. 当前判断（事实 + 推断/假设，置信度: 高）：
   - 在当前原型层级下，不建议把 `当前概览 / 工作集变化 / 三轮评测 / 历史记录` 直接提到页面最顶部。

4. 下一轮衔接点（事实）：
   - 若用户希望进一步增强“页签更显眼”，下一轮可优先尝试“放到中部主卡最上沿、紧贴标题区”的方案，而不是直接抬到整页顶栏。
